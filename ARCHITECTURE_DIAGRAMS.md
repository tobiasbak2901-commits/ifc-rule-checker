Rendering Flow (Deferred Pattern)\n\n```\nOperation Triggered\n(user rotates, geometry added, window resized, etc.)\n    вҶ“\nCall any render method:\n    в”ңв”Җ clear()\n    в”ңв”Җ fit_to_view()\n    в”ңв”Җ render()\n    в””в”Җ resizeEvent() / moveEvent()\n    вҶ“\nEach calls вҶ’ _schedule_render()\n    вҶ“\n_schedule_render() [NON-BLOCKING]\n    в”ңв”Җ Check: _pending_render already True?\n    в”Ӯ  в”ңв”Җ Yes вҶ’ Do nothing (already scheduled)\n    в”Ӯ  в””в”Җ No вҶ’ Continue\n    в”Ӯ\n    в”ңв”Җ Set _pending_render = True\n    в””в”Җ Schedule: QtCore.QTimer.singleShot(0, self._do_render)\n        в”Ӯ\n        в”ңв”Җ Timer with 0ms delay\n        в”ңв”Җ Queued in Qt event loop\n        в””в”Җ Will fire next event cycle\n    вҶ“\n[Function returns IMMEDIATELY - no rendering yet]\n    вҶ“\n[Qt processes other events...]\n    вҶ“\n[0ms timer fires]\n    вҶ“\n_do_render() [ACTUAL RENDERING]\n    в”ңв”Җ Set _pending_render = False\n    в””в”Җ Call render_window.Render()\n    вҶ“\n[Frame rendered once]\n```\n\n## 3.
